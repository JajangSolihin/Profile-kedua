CSS Merupakan kepanjangan dari Cascading Style Sheets. Pengertian CSS yang berasal dari Wikipedia adalah bahasa "pemrograman" yang digunakan untuk menggambarkan bentuk desain dan pemformatan pada bahasa yang menggunakan markup. Keuntungan CSS adalah kita tidak perlu menulis kode yang panjang dalam setiap tag HTML. Tapi dapat menghasilkan sebuah penggayaan untu website kita secara fleksibel.
